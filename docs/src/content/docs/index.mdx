**Starlight Quiz** adds interactive, self-marking quizzes to your Astro and Starlight documentation.

## Features

<CardGrid stagger>
  <Card title="✨ Simple markdown syntax">
    Write quizzes with GitHub-flavoured task lists — no new syntax to learn.
  </Card>
  <Card title="🎯 Multiple quiz types">
    Single choice, multiple choice and fill-in-the-blank.
  </Card>
  <Card title="⚡ Instant feedback">
    Per-answer feedback and visual correct/incorrect indicators.
  </Card>
  <Card title="📊 Progress & results">
    An aggregate results panel with score tiers and confetti.
  </Card>
  <Card title="💾 Results saved">
    Answers persist to the browser's local storage, surviving reloads and rebuilds.
  </Card>
  <Card title="🌐 Internationalisation">
    English, French, German and Spanish out of the box.
  </Card>
  <Card title="♿ Accessible">
    Real fieldsets, `aria-live` feedback and keyboard-safe auto-submit.
  </Card>
</CardGrid>
