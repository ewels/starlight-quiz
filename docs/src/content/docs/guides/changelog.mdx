Starlight Quiz follows [semantic versioning](https://semver.org/) and tracks changes with
[Changesets](https://github.com/changesets/changesets). Every notable change ships with a changeset that becomes a
changelog entry on release.

## Release notes

- **[GitHub Releases](https://github.com/ewels/starlight-quiz/releases)**: the notes for each published version.
- **[`CHANGELOG.md`](https://github.com/ewels/starlight-quiz/blob/main/packages/starlight-quiz/CHANGELOG.md)**: the
  same history in the package, generated by Changesets.

## Versioning

The package is published from `packages/starlight-quiz`. Install a specific version with your package manager, e.g.
`npm install starlight-quiz@latest`. See [Contributing](/starlight-quiz/guides/contributing/) for how changesets are
authored.
