Notable changes are recorded in [`CHANGELOG.md`](https://github.com/ewels/starlight-quiz/blob/main/CHANGELOG.md), with
in-progress work collected under an **Unreleased** heading until the next release.

## Release notes

- **[`CHANGELOG.md`](https://github.com/ewels/starlight-quiz/blob/main/CHANGELOG.md)**: the full history, also shipped
  inside the npm package.
- **[GitHub Releases](https://github.com/ewels/starlight-quiz/releases)**: the same notes per published version.

## Versioning

The package is published from `packages/starlight-quiz`. Install a specific version with your package manager, e.g.
`npm install starlight-quiz@latest`. See [Contributing](/starlight-quiz/guides/contributing/) for how releases are cut.
