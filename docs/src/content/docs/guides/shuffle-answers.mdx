---
title: Shuffle answers
description: Randomise the order of answers on every page load.
---

import { Quiz } from 'starlight-quiz/components';
import { Tabs, TabItem } from '@astrojs/starlight/components';

Set the `shuffle` prop to randomise the order of a choice quiz's answers. The shuffle happens in the browser on every
load, so readers see a different order each visit — useful to stop people memorising answer positions rather than the
content.

<Tabs syncKey="quiz-source">
<TabItem label="Preview">

<Quiz id="shuffle-demo" shuffle>
Which is the largest planet in the Solar System?

- [x] Jupiter
- [ ] Earth
- [ ] Mars
- [ ] Mercury

</Quiz>

</TabItem>
<TabItem label="Source">

```mdx title="quiz.mdx"
<Quiz shuffle>
Which is the largest planet in the Solar System?

- [x] Jupiter
- [ ] Earth
- [ ] Mars
- [ ] Mercury
</Quiz>
```

</TabItem>
</Tabs>

Reload the page a few times to see the order change. Correct-answer tracking is preserved regardless of display order,
and saved progress still restores correctly.

:::note
Shuffling applies to choice questions only; fill-in-the-blank questions have no answers to reorder.
:::
